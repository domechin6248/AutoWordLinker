# 📄 議案資料・自動リンクツール

このツールは、`議案書様式.docx` の中の文字と、同じ名前のフォルダ・ファイルを自動で結びつけ（ハイパーリンク）、一つのフォルダにまとめます。

## 📥 使い方（3ステップ）

1. **ダウンロードする**
   右上の緑色の「Code」ボタンを押し、「Download ZIP」を選んで、自分のパソコンに保存・解凍してください。

2. **資料を入れる**
   - `keikakusho` フォルダの中に、**「議案書様式.docx」** を入れます。
   - `link` フォルダの中に、**リンクさせたいフォルダ（01参考資料など）** をそのまま入れます。

3. **ボタンを押す**
   - Windowsの方：`リンク開始_Win.bat` をダブルクリック
   - Macの方：`リンク開始_Mac.app` をダブルクリック

## ✅ 完了
`output` フォルダの中に、議案書と同じ名前のフォルダができあがります。その中のWordはすでにリンク済みです！
